Projects

Robotics

Here there is a collection of all the projects that are strictly related to the design, development and functioning of robots:

Voice Controlled Mobile Robot for Indoor Applications   Slideshow

This thesis proposes a design from scratch for a small mobile robot controlled by voice using ROS. The results obtained show both the feasibility of using low power boards and some of their limitations. (Research publication in preparation)

Recharging Electric Cars with a Mobile Robot   Slideshow

The aim of this project was to prove the possibility to design an autonomous mobile robot whose task was to recharge electric cars in a parking lot. This was done in simulation, using the Gazebo and ROS.

Mobile Robot Self-Localization   Slideshow    GitHub   

For this project we worked using Matlab on data retrieved from a differential drive robot for its localization in a certain area by the means of the Extended Kalman filter (EKF).

Onychophora-Inspired Firefighting Robot   Slideshow    GitHub   

This project proposes a conceptual design for a firefighting robot suitable for indoor environments by taking inspiration from the biological world.

3RRR Planar Parallel Manipulator   Slideshow    GitHub   

This project concerns the design and optimization of a 3RRR planar parallel manipulator using MATLAB and CATIA

Point-to-Point Trajectory Optimization   Slideshow    GitHub   

For this project we worked to find the optimal trajectory for a simple model of a robot performing point-to-point navigation in a room with some obstacles using MATLAB.

Simple Swarm GitHub   

Simple simulation and visualization of swarms of differential drive robots following a leader. Developed using C++ and ROS.

Industrial Robot Programming GitHub   

Industrial Robot Programming using VAL II and V+ tested on a PUMA and a RX90 robots.

Modelling and Control of Manipulators GitHub   

This contains several projects regarding modelling and control of various manipulators using MATLAB, Simulink and ADAMS.

Engineering

Here there is a collection of all the projects that are related to all the other fields of engineering:

Solar Panel Fault Detection Method Based on Infrared Image Processing   Slideshow

My Chinese bachelor thesis that proposes a method to detect malfunctioning solar panels by processing thermal images using MATLAB.

Design of a Testing Machine for Flexure Hinges   Slideshow

My Italian bachelor thesis that proposes a design from scratch of a machine for the empirical characterization of flexure hinges.

Control of a Nonlinear System   Slideshow    GitHub   

The aim of this project was to implement control laws for a nonlinear system (based on the model of a PVTOL aircraft).

Multi-Body System’s Kinematic Analysis   Slideshow    GitHub   

For this project the kinematic analysis has been performed upon agiven mechanism using MATLAB. It has been then validated by comparing it to the ADAMS model of the same mechanism.

Signal Processing GitHub   

Several Signal processing projects developed using MATLAB.

Real Time Systems GitHub   

Several real-time systems' projects developed using C++ and OIL, and run using Trampoline RTOS on a Teensy board.
